# SQL Book Stores Data Analysis Project

## Project Overview
This project focuses on analyzing data from a Book Store database to extract meaningful business insights. By querying relational data across books, customers, and orders, the project helps understand sales patterns, customer behavior, and inventory performance.

## Key Insights & Business Questions Answered
- **Sales Performance**: Tracking total revenue generated from book sales over time.
- **Customer Segmentation**: Identifying top customers based on their order frequency and total spending.
- **Inventory & Popularity**: Finding the most popular book genres, top-selling authors, and low-stock items.
- **Order Tracking**: Analyzing order statuses and customer purchasing trends.

## Tech Stack Used
- **SQL / MySQL Workbench**: For writing complex relational queries, joins, aggregations, and data analysis.
- **Data Source**: Relational datasets containing detailed information about Books, Customers, and Orders.
